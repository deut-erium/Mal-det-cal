Usage information for training data:
(1) if the training files have already been filtered, do `python3 train.py`
(2) if the training files have not been filtered, do `python3 train.py filter <path to training data>`

Step (1) and (2) saves the trained model into "trained_model.pickle" file.


Usage information for testing  data:
(3) `python3 MalwareDetection.py <path to testing data>`

This generates the required "output.csv" file.
